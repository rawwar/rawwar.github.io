Plan is to learn Go Data Types and practise them.

- Basic types: int, float64, string, bool
- Composite types: arrays, slices, maps, structs
- Pointers and interfaces
- Type conversions and zero values
- Practice exercises: implement a simple struct with methods, use maps for counting, manipulate slices, etc.
